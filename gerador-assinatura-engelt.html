HTML — Cole no Thunderbird ou Outlook

Como instalar no Outlook:
1. Clique "Salvar para Outlook" — o arquivo já vem com o nome do colaborador
2. Clique "Abrir pasta de Assinaturas" e mova o arquivo baixado para lá
3. No Outlook: Arquivo → Opções → Email → Assinaturas → selecione → OK

Thunderbird: Menu ≡ → Configurações → sua conta → Assinatura → marcar "Usar HTML" → colar o código
